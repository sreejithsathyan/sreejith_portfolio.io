Introduction

The goal is to achieve an IoT system that calculates the consumption of energy depending on when an appliance is powered / disconnected throughout the day.  The status on or off of the system is tracked by a time series database (influxDB), the on / off information (not at a fixed time basis) is irregular. Regularly (say, each hour), node-red queries the time series for each device and calculates the run time for that device for the same duration of 1 hour. The operating time should then be converted and stored in a new time series in an energy value (Simple Multiplier).

NodeRed

Node-RED is a visual programming flow-based development tool originally developed by IBM to connect hardware devices, APIs and online services as part of the Internet of Things. Node-RED provides a web browser-based flow editor that can be used to create JavaScript functions. (NodeRed, 2019)

PostMan

Postman is a software for API testing Google Chrome plugin to communicate with HTTP APIs. It offers a friendly GUI to create requests and read responses. Postman's people also offer an add-on package called Jetpacks, which includes some automation tools and, most importantly, a testing library for JavaScript, It is developed by Started by Abhinav Asthana have headquarters Bengaluru India (Postman, 2019)

Azure Cloud

Microsoft Azure is a cloud computing platform developed by Microsoft to develop, check, deploy and manage applications and services via Microsoft's data centers. This offers software as a service (SaaS), platform as a service (PaaS) and infrastructure as a service (IaaS) and supports a range of programming languages, technologies and frameworks, including applications and systems common to Microsoft and third-party. (Azure, 2019)

Virtual Machine

Initially, a virtual machine was established by Popek and Goldberg as an effective, isolated duplicate of a real computer machine. Modern usage involves virtual machines that do not relate directly to any actual hardware. The physical, real-world hardware running the VM is generally referred to as the host. A host will simulate multiple visitors, each emulating specific hardware and operating systems. Most popular virtual machine platforms are Microsoft Azure, Amazon Web Services, VMware is a software which stimulate another operating system over one operating system. Like In windows its possible to run Linux full os with the help of VMware, Latest windows 10 version have built in virtual machine support for Windows Subsystem for Linux. (Virtual Machine, 2019)

Docker Container

Docker is a set of platforms as a service (PaaS) products that provide software in packages called containers using OS-level virtualization. Containers are separated from each other and contain their own code, libraries and configuration files; through well-defined channels, they may interact with each other. Both containers are powered by a single kernel operating system and are therefore lighter than virtual machines.  The system has free rates as well as premium levels. Docker Engine is the program that runs the containers. It launched in 2013 and is being built by Docker, Inc. (Docker, 2019)

InfluxDB

Influx DB is an open source time series database that Influx Data has created. It is written in Go and optimized in the field of operational monitoring, application measurements, the Internet of sensor data and real-time analytics for fast, high-availability storage and retrieval. Mainly used for Internet of Tings technologies (InfluxDb, 2019)

SSH

SSH port forwarding is a process in SSH for tunneling from client machine to server machine device ports, or vice versa. It can be used to add encryption to legacy applications, pass through firewalls, and it can be used by some system administrators and IT professionals to open backdoors from their home machines into the internal network. The opening of internet access to the internal network can also be abused by hackers and malware. (ssh, 2019)
